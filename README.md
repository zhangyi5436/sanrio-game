# 三丽鸥益智贪吃蛇 🎀

一款专为 6-10 岁儿童设计的益智 H5 小游戏，以经典贪吃蛇为核心玩法，融合三丽鸥卡通IP与 20 以内加减法算术训练，实现教育与娱乐结合。

## 游戏特色

- ✨ **三丽鸥卡通主题** - 5个经典角色可选：凯蒂猫、美乐蒂、玉桂狗、库洛米、布丁狗
- 🧮 **算数练习** - 随机生成20以内加减法，边玩边练
- 👨‍👦 **支持单/双人PK** - PC支持双人对战，手机适配单人触屏操作
- 📱 **全端一屏适配** - PC/手机完美适配，无滚动条，内容完整展示
- 👶 **儿童友好设计** - 柔和配色、圆角设计、卡通字体，操作简单

## 玩法说明

1. 选择游戏模式（PC端可选单人/双人，手机默认为单人）
2. 选择你喜欢的三丽鸥角色作为贪吃蛇头部
3. 控制贪吃蛇吃掉正确答案的金蛋
4. ✅ 答对：蛇身变长 + 得分，刷新题目
5. ❌ 答错：蛇身变短，不刷新题目
6. 蛇身长归零游戏结束，限时5分钟

## 操作方式

- **PC单人模式**：方向键 ↑↓←→ 控制
- **PC双人模式**：玩家1 (WSAD)，玩家2 (方向键)
- **手机触屏**：上下左右滑动控制

## 技术栈

- HTML5 + CSS3 + 纯 JavaScript
- Canvas 绘制游戏画面
- 原生实现，无需任何依赖

## 本地运行

```bash
# 直接用浏览器打开 index.html 即可
# 或者使用本地服务器
python3 -m http.server 8000
# 访问 http://localhost:8000
```
